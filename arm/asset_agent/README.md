# asset_agent (`workshop.arm.asset_agent`)

Busca multi-fonte por cena (laboratório), alinhada a `agents.asset_agent`.

## Como rodar (raiz do repositório)

```bash
python workshop/arm/asset_agent/search.py --scene "Scene 2: artist impression of a black hole with accretion disk, bright rings."
```

Chaves opcionais: `PEXELS_API_KEY`, `PIXABAY_API_KEY`, `UNSPLASH_ACCESS_KEY` (por exemplo em `workshop/.env`).
